# Enterprise Architecture with .NET & Entity Framework

This repository serves as a reference implementation for **N-Tier Architecture** in C#. 
It demonstrates the separation of concerns by isolating:
- **Entity Layer:** Database models.
- **Data Access Layer:** CRUD operations via Entity Framework.
- **Business Layer:** Logic and validation rules.
- **Presentation Layer:** User interface integration.

**Tech Stack:** C#, .NET Framework, SQL Server, Entity Framework.
